Advanced Mac Substitute Code Resources
======================================

Startup
-------

  * 'INIT' 31 (runs INITs from files)

User Interface
--------------

  * 'CDEF' 0 (push buttons, checkboxes, and radio buttons)
  * 'CDEF' 1 (scroll bars)
  * 'LDEF' 0 (vertical list of strings)
  * 'MDEF' 0 (standard menus)
  * 'WDEF' 0 (standard document windows and dialogs)
  * 'WDEF' 1 (round-cornered windows for DAs)
